Welcome to dynamite-sdk-lite's documentation!
=============================================

Dynamite SDK is the companion software development kit to DynamiteNSM_.

The main purpose of this SDK is provide a set of wrappers for working with the events generated by the NSM.

.. _DynamiteNSM: https://github.com/DynamiteAI/dynamite-nsm
